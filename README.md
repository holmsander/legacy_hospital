This is a simulation of how a messy back end soultion for some hospital could be integrated into modern technology.
The main features it offers is:
Given a doctors note from a patients visit it automatically takes all the notes and understands it, also understands what local codes this hospital might use for different
diagnoses, then automatically insert relevant data into the backend without needing someone to manually enter all the details. Before inserting however, the AI Agent will ask for doctors confirmation:
Does this seem correct? And if not lets doctor adjust small things.
